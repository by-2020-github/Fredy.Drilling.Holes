# 回零流程说明（当前实现）

> 对应源文件：`HAL/Motion/MotionAdt8940.cs`  
> 更新时间：2025-07  
> 执行方式：全异步（`async/await + CancellationToken`），取消即触发急停所有轴

---

## 一、总入口

```
HomeAsync(axisNo, wait, cancellationToken)
        │
        ├─ axisNo == 3 ──→ HomeZAxisAsync
        └─ axisNo == 1/2 → HomeXYAxisAsync
```

---

## 二、XY 轴回零总流程

```mermaid
graph TD
    START([开始 XY 回零]) --> RETRY_LOOP[进入重试循环 最多3次]

    RETRY_LOOP --> CHECK_CANCEL{CancellationToken\n已取消?}
    CHECK_CANCEL -- 是 --> EMERG[EmergencyStopAllEnabledAxesAsync\n急停所有轴 1/2/3\n记录当前位置日志]
    EMERG --> THROW_CANCEL([抛出 OperationCanceledException])

    CHECK_CANCEL -- 否 --> MECH[HomeXYMechanicalAsync\n机械回零]
    MECH --> MECH_OK{成功?}
    MECH_OK -- 否/超时 --> CATCH[catch Exception\nStopAsync 停当前轴]
    CATCH --> RETRY_CHECK{retry < 3?}
    RETRY_CHECK -- 是 --> RETRY_LOOP
    RETRY_CHECK -- 否 --> SAFE_STOP[SafeStopAllAxesAsync\n减速停止所有轴]
    SAFE_STOP --> THROW_FAIL([抛出 InvalidOperationException\n回零失败])

    MECH_OK -- 是 --> GRATING_CHECK{IsGratingHome?}
    GRATING_CHECK -- 否 --> RESET_POS[ResetPosition\n清零坐标]
    GRATING_CHECK -- 是 --> MODE_CHECK{回零模式}
    MODE_CHECK -- IsLatch --> LATCH[HomeXYByLatchAsync\n锁存回零]
    MODE_CHECK -- IsIoHome --> IO[HomeXYByGratingIoAsync\nIO光栅回零]
    LATCH --> RESET_POS
    IO --> RESET_POS
    RESET_POS --> SUCCESS([回零成功])

    style START fill:#90EE90
    style SUCCESS fill:#90EE90
    style THROW_FAIL fill:#FFB6C1
    style THROW_CANCEL fill:#FFA07A
    style EMERG fill:#FFA07A
    style SAFE_STOP fill:#FFB6C1
```

## 三、机械回零详细流程（HomeXYMechanicalAsync）

> 传感器激活电平由 `HomingPort.IsLowLevelActive` 决定：  
> - `IsLowLevelActive = true` → `activeLevel = 0`  
> - `IsLowLevelActive = false` → `activeLevel = 1`

```mermaid
graph TD
    M_START([HomeXYMechanicalAsync 开始]) --> READ_LEVEL[读取当前机械开关电平]
    READ_LEVEL --> LEVEL_CHECK{当前电平\n== activeLevel?}

    LEVEL_CHECK -- 是\nLEFT BRANCH --> LEFT_MOVE[向正方向 dir=1 连续运动\n等待电平变为 inactiveLevel\n超时 HomeTimeoutMs]
    LEFT_MOVE --> LEFT_STOP[停止]
    LEFT_STOP --> LEFT_BACKOFF[向正方向移动退让脉冲\nHomeBackoffPulse]

    LEVEL_CHECK -- 否\nRIGHT BRANCH --> RIGHT_MOVE[向负方向 dir=0 连续运动\n等待电平变为 inactiveLevel\n超时 HomeTimeoutMs]
    RIGHT_MOVE --> RIGHT_STOP[停止]
    RIGHT_STOP --> RIGHT_BACKOFF[向正方向移动退让脉冲\nHomeBackoffPulse]

    LEFT_BACKOFF --> SLOW_APPROACH
    RIGHT_BACKOFF --> SLOW_APPROACH

    SLOW_APPROACH[慢速向负方向 dir=0 连续运动\nSlowHomeSpeed / SlowHomeAcceleration\n等待电平变为 activeLevel\n超时 HomeTimeoutMs] --> SLOW_STOP[停止]
    SLOW_STOP --> M_END([机械回零完成])

    style M_START fill:#90EE90
    style M_END fill:#90EE90
```

**关键参数说明：**

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `HomeSearchSpeed` | 粗找零速度（脉冲/s） | 3000 |
| `HomeBackoffPulse` | 离开传感器后的退让脉冲数 | 200 |
| `SlowHomeStartSpeed` | 慢速起始速度 | 100 |
| `SlowHomeSpeed` | 慢速运行速度 | 500 |
| `SlowHomeAcceleration` | 慢速加速度 | 1000 |
| `HomeTimeoutMs` | 单步超时（ms） | 10000 |

## 五、锁存模式回零（HomeXYByLatchAsync）

```mermaid
graph TD
    L_START([锁存模式回零开始]) --> CLR_LATCH[清除锁存状态\nClearLockStatusAsync]
    CLR_LATCH --> SET_LATCH[设置锁存模式\nSetLockPositionModeAsync mode=1,regi=0,logical=1]
    SET_LATCH --> LATCH_MOVE[向负方向 dir=0 连续运动\nGratingHomeSpeed]
    LATCH_MOVE --> WAIT_LATCH[轮询等待硬件锁存信号\n超时 HomeTimeoutMs]
    WAIT_LATCH --> LATCH_GOT{锁存信号\n触发?}
    LATCH_GOT -- 超时 --> LATCH_TIMEOUT([抛出 TimeoutException])
    LATCH_GOT -- 是 --> LATCH_STOP[StopAsync 停止轴]
    LATCH_STOP --> READ_LATCH_POS[读取锁存脉冲位置\nGetLockPositionPulse]
    READ_LATCH_POS --> CALC_DIST[计算移动距离\nlockPosition - currentPosition]
    CALC_DIST --> MOVE_TO[MoveRelativePulseAsync\n移动到锁存位置\n等待到位]
    MOVE_TO --> FINALLY[finally: 清除锁存模式\nSetLockPositionMode mode=0\nClearLockStatus]
    FINALLY --> L_END([锁存模式回零完成])

    style L_START fill:#90EE90
    style L_END fill:#90EE90
    style LATCH_TIMEOUT fill:#FFB6C1
```

## 六、Z 轴回零流程（HomeZAxisAsync）

```mermaid
graph TD
    Z_START([Z轴回零开始]) --> Z_READ[读取当前 ZLimitPort 电平]
    Z_READ --> Z_CHECK{当前电平\n== activeLevel?}

    Z_CHECK -- 是 --> Z_AWAY[向反方向运动\nZHomeTowardPositiveDirection ? dir=0 : dir=1\n等待离开激活态\n超时 HomeTimeoutMs]
    Z_AWAY --> Z_TOWARD

    Z_CHECK -- 否 --> Z_TOWARD[向目标方向运动\nZHomeTowardPositiveDirection ? dir=1 : dir=0\n等待到达激活态\n超时 HomeTimeoutMs]
    Z_TOWARD --> Z_STOP[停止]
    Z_STOP --> Z_RESET[ResetPosition 清零坐标]
    Z_RESET --> Z_LIFT_CHECK{ZHomeLiftPulse > 0?}
    Z_LIFT_CHECK -- 是 --> Z_LIFT[MoveRelativePulseAsync\n向上抬起 ZHomeLiftPulse 脉冲\n等待到位]
    Z_LIFT --> Z_END([Z轴回零完成])
    Z_LIFT_CHECK -- 否 --> Z_END

    style Z_START fill:#90EE90
    style Z_END fill:#90EE90
```

**关键参数：**

| 参数 | 说明 |
|------|------|
| `ZLimitPort` | Z轴机械零位 IO 端口及有效电平 |
| `ZHomeTowardPositiveDirection` | 回零向正方向运动（true）或负方向（false） |
| `ZHomeLiftPulse` | 回零完成后抬起脉冲数（0=不抬起） |
| `HomeSearchSpeed` | Z轴找零速度 |

## 七、紧急停止处理

```mermaid
graph TD
    CANCEL[CancellationToken 触发] --> EMERG_ALL[EmergencyStopAllEnabledAxesAsync]
    EMERG_ALL --> SUDDEN1[adt8940a1_sudden_stop 轴1]
    EMERG_ALL --> SUDDEN2[adt8940a1_sudden_stop 轴2]
    EMERG_ALL --> SUDDEN3[adt8940a1_sudden_stop 轴3]
    SUDDEN1 & SUDDEN2 & SUDDEN3 --> SAVE_POS[读取并日志记录各轴当前位置]
    SAVE_POS --> RECORD[RecordNativeCall 记录事件]
    RECORD --> THROW([重新抛出 OperationCanceledException])

    style CANCEL fill:#FFA07A
    style THROW fill:#FFA07A
```

## 八、HomingOptions 配置参数总览

| 参数 | 类型 | 说明 |
|------|------|------|
| `HomeSearchSpeed` | int | 粗找零速度（脉冲/s） |
| `IsIoHome` | bool | 启用 IO 光栅回零 |
| `IsLatch` | bool | 启用硬件锁存回零 |
| `IsGratingHome` | bool | 启用光栅尺回零（总开关） |
| `XLimitPort` | HomingPort | X轴机械零位端口+有效电平 |
| `YLimitPort` | HomingPort | Y轴机械零位端口+有效电平 |
| `ZLimitPort` | HomingPort | Z轴机械零位端口+有效电平 |
| `XGratingPort` | HomingPort | X轴光栅零位端口+有效电平 |
| `YGratingPort` | HomingPort | Y轴光栅零位端口+有效电平 |
| `HomeTimeoutMs` | int | 单步等待超时（ms），默认 10000 |
| `HomeBackoffPulse` | int | 退让脉冲数，默认 200 |
| `ZHomeLiftPulse` | int | Z轴回零后抬起脉冲数，默认 0 |
| `ZHomeTowardPositiveDirection` | bool | Z轴回零方向，默认 false（负方向） |
| `SlowHomeStartSpeed` | int | 慢速起始速度，默认 100 |
| `SlowHomeSpeed` | int | 慢速运行速度，默认 500 |
| `SlowHomeAcceleration` | int | 慢速加速度，默认 1000 |
| `GratingHomeStartSpeed` | int | 光栅找零起始速度，默认 500 |
| `GratingHomeSpeed` | int | 光栅找零速度，默认 2000 |
| `GratingHomeAcceleration` | int | 光栅找零加速度，默认 2000 |
