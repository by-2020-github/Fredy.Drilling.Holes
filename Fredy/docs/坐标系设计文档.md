# 坐标系设计文档

> 对应实现：`Fredy/Services/CoordinateService.cs`

---

## 1. 概述

本系统涉及两套物理坐标和一套工件逻辑坐标，通过 `CoordinateService` 统一管理转换与校准。

---

## 2. 坐标概念定义

> **重要说明**：相机和冲针共用同一套 XY 运动轴（机械坐标系），两者并非独立坐标系，
> 只是由于安装位置不同，在同一时刻两者所处的机械坐标值相差一个固定偏移量。

| 概念 | 英文标识 | 原点定义 | 主要用途 |
|---|---|---|---|
| **机械坐标（冲针位）** | Machine | 运动控制卡物理原点，由回零确定 | 运动控制目标位置，直接下发给运动卡 |
| **机械坐标（相机位）** | Machine @ Camera | 与冲针位共用同一套坐标轴，相差固定偏移 `CameraToPunchOffset` | 相机视觉采集时记录的机械坐标 |
| **工件/冲针坐标系** | Workpiece | 冲针对准工件圆心时的机械坐标为原点 | Recipe 孔位坐标存储与编辑 |

---

## 3. 坐标转换关系

```
相机采集的机械坐标
       │
       │  减去 CameraToPunchOffset
       ▼
冲针等效的机械坐标
       │
       │  减去 WorkpieceCenter
       ▼
工件/Recipe 坐标
```

### 3.1 工件坐标系 ↔ 机械坐标系（冲针位，纯平移）

```
机械坐标(冲针) = 工件坐标 + WorkpieceCenter
工件坐标       = 机械坐标(冲针) - WorkpieceCenter
```

`WorkpieceCenter` = 冲针对准工件圆心时的机械坐标，由**校准2**计算得出：

```
WorkpieceCenter = FittedCircleCenter(相机位) - CameraToPunchOffset
```

### 3.2 相机位机械坐标 ↔ 冲针位机械坐标（纯平移）

```
机械坐标(冲针) = 机械坐标(相机) - CameraToPunchOffset
机械坐标(相机) = 机械坐标(冲针) + CameraToPunchOffset
```

`CameraToPunchOffset` 由**校准1**确定。

### 3.3 相机位机械坐标 → 工件坐标（两步合并）

```
工件坐标 = 机械坐标(相机) - CameraToPunchOffset - WorkpieceCenter
```

---

## 4. 校准方法

> **校准顺序**：校准1 必须先于校准2 完成。

### 4.1 校准1 — 相机偏移校准（`CalibrateCameraToPunchOffset`）

**目标**：确定相机视野中心相对于冲针的固定机械坐标偏移量。

**操作步骤**：
1. 在任意位置执行冲孔，记录此时冲针的机械坐标 **P_punch**。
2. 移动平台，直至相机视野中心与该孔重合，记录此时机械坐标 **P_camera**。
3. 调用 `CalibrateCameraToPunchOffset(P_punch, P_camera)`。

**计算公式**：
```
CameraToPunchOffsetX = P_camera.X - P_punch.X
CameraToPunchOffsetY = P_camera.Y - P_punch.Y
```

**结果写入**：`Calibration.CameraToPunchOffsetX / CameraToPunchOffsetY`

---

### 4.2 校准2 — 工件坐标系原点校准（`CalibrateWorkpieceByEdgePoints`）

**目标**：确定冲针对准工件圆心时的机械坐标（即工件坐标系原点 `WorkpieceCenter`）。

**前提**：校准1已完成。

**操作步骤**：
1. 移动平台，使相机视野中心对准工件圆形边缘不同位置（**至少 3 点**，建议 5 点以上均匀分布），
   每次记录**相机所处的机械坐标**。
2. 调用 `CalibrateWorkpieceByEdgePoints(edgePoints, out radius)`。
3. 内部流程：
   - 使用最小二乘圆拟合（`CircleFitter.FitCircle`）计算相机视野对准圆心时的机械坐标 `FittedCenter`。
   - 减去 `CameraToPunchOffset` 换算为冲针等效坐标，存入 `WorkpieceCenter`。

**计算公式**：
```
FittedCenter   = CircleFitter.FitCircle(边缘点机械坐标)   // 相机对准圆心时的机械坐标
WorkpieceCenter = FittedCenter - CameraToPunchOffset        // 冲针对准圆心时的机械坐标
```

**结果写入**：`Calibration.WorkpieceCenterX / WorkpieceCenterY`，同时输出拟合半径供核验。

**注意事项**：
- 采集的边缘点是**相机中心**所处的机械坐标，不是冲针坐标。
- 点数越多、分布越均匀，拟合精度越高。
- 若采集点共线，拟合将失败并抛出异常。

## 5. 校准参数模型（`CoordinateCalibration`）

| 字段 | 类型 | 说明 |
|---|---|---|
| `WorkpieceCenterX` | double | 冲针对准工件圆心时的机械坐标 X（工件坐标系原点） |
| `WorkpieceCenterY` | double | 冲针对准工件圆心时的机械坐标 Y（工件坐标系原点） |
| `CameraToPunchOffsetX` | double | 相机中心相对冲针的 X 偏移（机械坐标系，= 相机位 − 冲针位） |
| `CameraToPunchOffsetY` | double | 相机中心相对冲针的 Y 偏移（机械坐标系，= 相机位 − 冲针位） |
| `CameraToWorkpieceRotationRad` | double | 相机轴向与机械轴向的旋转修正角（弧度），默认 0 |
| `IsWorkpieceCenterCalibrated` | bool | 工件圆心是否已校准（只读） |
| `IsCameraOffsetCalibrated` | bool | 相机偏移是否已校准（只读） |

## 6. API 速查

```csharp
// ── 最常用：工件坐标 → 冲针机械坐标（下发运动指令）──────────────────────
Point2D machine = coordinateService.WorkpieceToMachine(new Point2D(x, y));

// 冲针机械坐标 → 工件坐标
Point2D workpiece = coordinateService.MachineToWorkpiece(new Point2D(x, y));

// ── 相机坐标（相机位机械坐标）转换 ──────────────────────────────────────
// 相机位机械坐标 → 冲针等效机械坐标
Point2D machine = coordinateService.CameraToMachine(cameraMachinePoint);

// 冲针机械坐标 → 相机位机械坐标
Point2D cameraPos = coordinateService.MachineToCamera(machinePoint);

// 相机位机械坐标 → 工件坐标（两步合并：减Offset再减WorkpieceCenter）
Point2D workpiece = coordinateService.CameraToWorkpiece(cameraMachinePoint);

// 工件坐标 → 相机位机械坐标
Point2D cameraPos = coordinateService.WorkpieceToCamera(workpiecePoint);

// ── 校准 ─────────────────────────────────────────────────────────────────
// 校准1：计算相机相对冲针的偏移
coordinateService.CalibrateCameraToPunchOffset(punchMachinePos, cameraMachinePos);

// 校准2：通过边缘点拟合工件圆心（需先完成校准1）
coordinateService.CalibrateWorkpieceByEdgePoints(edgePointsAtCamera, out double radius);

// 直接设置已知工件圆心（冲针等效机械坐标）
coordinateService.SetWorkpieceCenter(new Point2D(cx, cy));
```

## 7. 典型工作流程

```
回零
  │
  ▼
【校准1】在任意位置冲一个孔，记录冲针机械坐标 P_punch
         移动平台使相机视野中心对准该孔，记录机械坐标 P_camera
         CalibrateCameraToPunchOffset(P_punch, P_camera)
         → CameraToPunchOffset 已知
  │
  ▼
【换新工件后】
【校准2】移动相机对准工件边缘 ≥3 点，每次记录相机所处机械坐标
         CalibrateWorkpieceByEdgePoints(edgePoints)
         → FittedCenter(相机位) − CameraToPunchOffset = WorkpieceCenter(冲针等效)
  │
  ▼
保存 CoordinateCalibration 到 AppConfig（持久化）
  │
  ▼
生产：Recipe 孔位（工件坐标）
      + WorkpieceCenter
      = 冲针机械坐标
      → 下发运动指令
```

## 8. 扩展说明

- **旋转角修正**：`CameraToWorkpieceRotationRad` 默认为 0，适用于相机安装方向与机械轴方向完全对齐的情况。若存在安装偏转，可扩展 Kabsch 最优旋转算法并在此处赋值。
- **持久化**：`CoordinateCalibration` 为纯数据类，可直接通过 `ConfigService` 序列化为 JSON 与 `AppConfig` 一起持久化；校准1结果（Offset）一般长期有效，校准2结果（WorkpieceCenter）每次换工件后需重新执行。
- **线程安全**：`CoordinateService` 本身不含状态锁，若多线程共享，调用方需在外部加锁或使用 DI 单例模式。
